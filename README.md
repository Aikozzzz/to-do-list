# To‑Do List (React + Vite)

Minimal to-do list app built with React and Vite.

## Setup

Install dependencies and start the dev server:

bash
npm install
npm run dev


Open the printed local URL in your browser.

Features:
- Add tasks
- Toggle complete
- Delete tasks
- Saves to `localStorage`
